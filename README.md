# Praxis 5589 Study Center

Static site designed for GitHub Pages.

## Deploy

1. Copy `index.html` to the root of your repository.
2. In GitHub, open **Settings → Pages**.
3. Deploy from your main branch and repository root, or use GitHub Actions.
4. No build command or dependencies are required.

All navigation uses URL hashes, so the site works under both user/organization pages and project pages.
